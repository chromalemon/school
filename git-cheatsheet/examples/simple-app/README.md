# Simple App

This is a simple Python application designed to help users practice their Git skills. It serves as an example project where you can learn how to use Git for version control and collaboration.

## Purpose

The purpose of this application is to provide a hands-on experience with Git, allowing users to track changes, create branches, and collaborate with others effectively.

## How to Run

To run this application, follow these steps:

1. Clone the repository to your local machine:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd simple-app
   ```

3. Run the application:
   ```
   python src/main.py
   ```

## How to Contribute

We welcome contributions to this project! If you would like to contribute, please follow these steps:

1. Fork the repository.
2. Create a new branch for your feature or bug fix:
   ```
   git checkout -b my-feature-branch
   ```
3. Make your changes and commit them:
   ```
   git commit -m "Description of my changes"
   ```
4. Push your changes to your fork:
   ```
   git push origin my-feature-branch
   ```
5. Create a pull request to the main repository.

Thank you for your interest in contributing to this project!
